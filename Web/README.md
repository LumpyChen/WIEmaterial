# Web前端方向新人培养流程

## 预阶段: 准备

* 明确自己的目标
* 学会解决问题
* 学习`Markdown` -- 简单而快速地写出具有结构的文本
* 学习`Git` -- 基础而强大的代码管理工具
* 学习使用`Github` -- 保存，分享，交流
* 掌握一个文本编辑器`Atom`/`Sublime Text`的使用方式

## 第一阶段: 元素与布局

* `HTML`语法以及常用元素种类
* `HTML`掌握引入各种元素的方式
* `CSS`掌握语法与使用方法
* `CSS`盒子模型与常用属性
* `CSS`深入理解`position`，研究居中的写法
* `CSS`深入理解`float`，研究布局的写法
* `CSS`对`CSS3`的变形进行与进行理解
* `CSS`学会响应式布局，同时对移动端页面进行处理
* `CSS`学习`BootStrap`
